# 掌上忻师 - 在线报到系统 #

该系统包含以下内容：

1. 高校新生扫码报到
2. 辅导员实时查看本人引导报到情况
3. 辅导员实时查看全体辅导员引导报到情况
4. 管理组实时查看分考试类型分专业报到情况

X: 版本号, Y: 修订号, Z: 迭代次数
